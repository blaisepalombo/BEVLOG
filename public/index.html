Frontend for the Energy Drink API

Energy Drink Tracker

A simple frontend built to work with a custom Node, Express, and MongoDB API. Log drinks, review entries, and manage your data through a cleaner interface.

What this frontend does

  • Displays drink entries from the API
  • Lets you add, edit, and delete drinks
  • Uses filtering, sorting, and stats endpoints
